# Portfolio Pessoal - Lucas Domingos Costa

Este é o código-fonte do meu portfolio pessoal. O objetivo deste projeto é apresentar minha trajetória, tecnologias com as quais trabalho e meus projetos principais de forma limpa, direta e profissional.

O site foi construído do zero, focando em performance e em uma interface que se adapta a qualquer dispositivo (computador ou celular).

---

##  Tecnologias

* **HTML5**: Estruturação semântica do conteúdo.
* **CSS3**: Estilização moderna utilizando variáveis, Flexbox e Grid.
* **JavaScript**: Lógica para animações, contadores e alternância de temas.
* **LinkedIn Badge API**: Integração oficial para exibição do perfil profissional.

---

##  Funcionalidades

- **Modo Dark & Light**: Alternância de tema para melhor conforto visual.
- **Animações de Scroll**: Elementos que surgem suavemente conforme a navegação.
- **Contadores Animados**: Exibição dinâmica de estatísticas e experiência.
- **Design Responsivo**: Layout que se ajusta automaticamente a diferentes tamanhos de tela.
